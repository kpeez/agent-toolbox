# Spec Templates

File templates for `/spec new`. Write each to its target path under `specs/`.

<templates>

<template file="specs/AGENTS.md">
# Specs

Each spec lives in `specs/<feature>/`.

Read order:

1. `AGENTS.md`
2. `PLAN.md`
3. `SPEC.md`
4. `STATUS.md`
5. `examples/RUN_LOG.md`

Do not manually maintain an index. Derive summaries by scanning
`specs/*/STATUS.md`. The project-level `specs/STATUS.md` file is generated by
`spec-status.py`; do not edit it by hand.
</template>
